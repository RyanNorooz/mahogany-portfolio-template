# Portfolio Website Template

Portfolio Template built with vue.js

## Run

```bash
# get pnpm
npm i -g pnpm

# install deps
pnpm i

# run dev server
pnpm dev

# build static site
pnpm build
```

## Pre-packed

### UI Frameworks

- [Bootstrap 5](https://github.com/twbs/bootstrap) ([Bootstrap](https://getbootstrap.com/)) - Build fast, responsive sites with Bootstrap

### Icons

- [FontAwesome](https://fontawesome.com/) - use amazimg icons the best icon sets

### Plugins

- [Vue Router](https://github.com/vuejs/vue-router)
  - [`vite-plugin-pages`](https://github.com/hannoeru/vite-plugin-pages) - file system based routing
- [Pinia](https://pinia.esm.dev) - Intuitive, type safe, light and flexible Store for Vue using the composition api
- [`unplugin-vue-components`](https://github.com/antfu/unplugin-vue-components) - components auto import
- [`unplugin-auto-import`](https://github.com/antfu/unplugin-auto-import) - Directly use Vue Composition API and others without importing
- [VueUse](https://github.com/antfu/vueuse) - collection of useful composition APIs
- [`@vueuse/head`](https://github.com/vueuse/head) - manipulate document head reactively

### Coding Style

- Use Composition API with [`<script setup>` SFC syntax](https://github.com/vuejs/rfcs/pull/227)
- [Prettier](https://prettier.io/) single quotes, no semi.
